# SW-AugmentedReality-Project
Το repository αυτό θα περιλαμβάνει όλα τα στοιχεία που αφορούν το project για το μάθημα της "Τεχνολογίας Λογισμικού" του ΣΤ'Εξαμήνου, του Τμήματος Πληροφορικής του Ιόνιου Πανεπιστημίου. Στο συγκεκριμένο project θα προχωρήσω σε προσθήκες και αλλαγές στο Εργαλείο επαύξησης του επιτραπέζιου παιχνιδιού D&amp;D, που ξεκίνησα στο πλαίσιο του μαθήματος "Πολυμέσα".
